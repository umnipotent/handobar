메뉴바에서 바로 확인하고 조작

Codex, Claude Code, Antigravity 잔여량을 메뉴바에 압축해 보여주고, 컨텍스트 메뉴에서 필요한 동작을 바로 실행합니다.

  • 선택한 도구의 잔여 퍼센트를 메뉴바 아이콘에 표시
  • 상세 패널 열기, 즉시 갱신, 앱 종료를 컨텍스트 메뉴로 실행
  • 패널을 열지 않아도 세션 상황을 빠르게 확인

지원 도구

자주 쓰는 AI 개발 도구의 남은 한도를 같은 화면에서 비교합니다.

Claude Code

5시간 한도, 주간 한도, 요금제 정보를 함께 표시합니다.

Codex

5시간 세션, 주간 스냅샷, 요금제 정보를 함께 표시합니다.

Antigravity

Gemini와 Not-Gemini 그룹, 추천 모델 칩, 요금제 정보를 함께 표시합니다.

핵심 기능

백그라운드에 상주하며 필요한 정보만 빠르게 확인하도록 구성했습니다.

  • 한도별 카드와 리셋 시각 표시
  • provider별 요금제 배지 표시
  • 60%↓ 주황, 40%↓ 빨강 경고
  • 트레이에 표시할 도구와 윈도우 선택
  • 카드 접기, 모델 칩 토글, 메모 지원

설치 안내

GitHub Releases에서 universal .dmg를 받아 Intel과 Apple Silicon Mac에 설치할 수 있습니다. 현재 ad-hoc 서명 및 미공증 빌드이므로 다운로드한 앱의 첫 실행은 Gatekeeper가 차단합니다.

최신 릴리스 정보는 페이지 로드 후 자동으로 확인합니다.

  • 시스템 설정 → 개인정보 보호 및 보안에서 아래로 스크롤한 뒤 "'HandoBar'이(가) 차단되었습니다" 옆 "그래도 열기"를 클릭하세요.
  • 또는 터미널에서 xattr -dr com.apple.quarantine /Applications/HandoBar.app를 실행하세요. 앱 경로가 다르면 맞게 조정해야 합니다.
  • 직접 pnpm tauri build로 빌드한 앱은 다운로드 앱이 아니므로 차단되지 않습니다. 차단 화면에서 "휴지통으로 이동"은 누르지 마세요.
